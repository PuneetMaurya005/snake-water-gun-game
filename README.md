# 🐍 Snake Water Gun Game

A classic Snake, Water, Gun game built in Python where you play 
against the computer in the terminal.

## 🎮 How to Play
- Snake beats Water 🐍
- Water beats Gun 💧
- Gun beats Snake 🔫

## ▶️ How to Run
1. Make sure Python is installed
2. Run the file:
bashpython swg.py
3. Enter your choice:
- `s` for Snake
- `w` for Water
- `g` for Gun

## 🧠 How It Works
- Computer randomly picks Snake, Water or Gun
- Your choice is compared with computer's choice
- Result is displayed instantly — Win, Lose or Draw!
